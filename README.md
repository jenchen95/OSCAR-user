## Installation

You can install OSCAR-user directly from GitHub:

```bash
# Using uv (faster, recommended)
uv pip install git+https://github.com/bq-zhu/OSCAR-user.git

# Or using pip
pip install git+https://github.com/bq-zhu/OSCAR-user.git
```

## Usage (Default Mode)
```python
import oscar
oscar.run()
```
## To uninstall
```bash
# Using uv
uv pip uninstall oscar

# Using pip
pip uninstall oscar
```
